nb - Notes & Todos CLI
======================

A plaintext-first command-line tool for managing notes and todos in markdown files.

**Key features:**

- Daily notes with automatic date-based organization
- Todo extraction from markdown checkboxes with rich metadata
- Multiple notebooks including external directories
- Hybrid search (semantic + keyword) powered by localvectordb
- Interactive terminal UI and web viewer
